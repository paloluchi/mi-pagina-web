¿Qué es un Sistema Operativo?

Un sistema operativo es un software que actúa como intermediario entre el usuario y el hardware, proporcionando un entorno para la ejecución de programas. Gestiona recursos como el procesador, la memoria, dispositivos de entrada/salida y almacenamiento, permitiendo a los usuarios interactuar con el sistema sin preocuparse por los detalles técnicos del hardware.

Componentes del Sistema de Computación

  • Hardware: UCP, memoria, dispositivos de E/S.
  • Sistema Operativo: Administra recursos y comunicación entre dispositivos.
  • Programas de Aplicación: Compiladores, sistemas de bases de datos.
  • Usuarios: Personas que interactúan con el sistema.

Funciones de los Sistemas Operativos

  • Configurar el equipo físico.
  • Interpretar comandos del usuario.
  • Controlar el hardware y la ejecución de programas.
  • Administrar el sistema de archivos y la memoria.
  • Controlar la secuencia de procesos.

Tipos de Sistemas Operativos

  • Por Número de Usuarios:
    • Monousuario: Solo un usuario a la vez (ej. MS-DOS).
    • Multiusuario: Más de un usuario simultáneamente (ej. Unix).
  • Por Número de Tareas:
    • Monotarea: Una tarea a la vez.
    • Multitarea: Varias tareas al mismo tiempo (ej. Windows).
  • Por Número de Procesadores:
    • Uniproceso: Maneja un procesador (ej. DOS).
    • Multiproceso: Utiliza varios procesadores (ej. Linux).

Arquitectura de un Sistema Operativo

  • Núcleo (Kernel): Comunicación directa con el hardware.
  • Servicios: Gestión de recursos y operaciones básicas.
  • Interfaz de Usuario (Shell): Interacción directa con el usuario.
