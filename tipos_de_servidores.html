Tipos de Redes

Red Doméstica

Wi-Fi: Dispositivos como teléfonos, notebooks y consolas conectados a un router de forma inalámbrica, permitiendo la movilidad dentro del hogar.

Cableada: Conexión de computadoras y dispositivos directamente al router mediante cables Ethernet, ofreciendo una mayor estabilidad y velocidad de conexión.

Transmisión Móvil: Los teléfonos se conectan a antenas (2G, 3G, 4G, 5G) que vinculan con nodos de la empresa telefónica y centrales de proveedores de Internet, permitiendo la conectividad fuera del hogar.

Internet Satelital: Utiliza antenas parabólicas para recibir y enviar señales a satélites, permitiendo el acceso a Internet en áreas remotas donde otros métodos no están disponibles.

Cajas Distribuidoras: Distribuyen Internet de ISPs (Proveedores de Servicios de Internet) en diferentes zonas, permitiendo que múltiples hogares se conecten a la red.

Redes Empresariales y Data Centers

Data Centers: Espacios físicos que albergan numerosas computadoras y servidores, encargados de almacenar y gestionar grandes volúmenes de información. Ejemplos: Datacom, GigaRed, Claro.

Redes Empresariales: Redes diseñadas para conectar computadoras, servidores y otros dispositivos dentro de una empresa, asegurando la comunicación y el intercambio de datos de manera segura.

Tipos de Servidores

Servidor FTP

Descripción: Permite la transferencia de archivos entre computadoras a través de la red de forma segura, utilizando el protocolo FTP (File Transfer Protocol). Es comúnmente utilizado para subir y descargar archivos en sitios web y servidores remotos.

Ventaja: Facilita la transferencia de grandes volúmenes de datos y permite controlar el acceso mediante autenticación de usuarios.

Desventaja: Puede ser vulnerable si no se implementan medidas de seguridad adicionales, como el uso de FTP seguro (FTPS) o SFTP.

Servidor Proxy

Descripción: Actúa como intermediario entre el usuario y el servidor final, ocultando la dirección IP del usuario y mejorando la seguridad. También se utiliza para acceder a contenido restringido y optimizar el uso de ancho de banda.

Ventaja: Mejora la privacidad y la seguridad de la navegación, además de permitir el acceso a contenidos bloqueados o restringidos.

Desventaja: Puede reducir la velocidad de conexión si no está bien configurado o si la capacidad del servidor es limitada.

Servidores Web

Descripción: Almacenan y entregan contenido web a los usuarios a través de Internet. Son fundamentales para la existencia de sitios web, aplicaciones web y servicios en la nube.

Tipos:

  • Hosting Compartido: Varios sitios web comparten el mismo servidor, lo que reduce costos, pero puede limitar los recursos asignados a cada sitio.
  • Hosting Dedicado: Un servidor exclusivo para un solo cliente, ofreciendo mayor control, rendimiento y personalización de la configuración del servidor.

Ventaja: Adaptable a las necesidades del cliente, desde opciones económicas con hosting compartido hasta soluciones personalizadas con hosting dedicado.

Desventaja: El costo de un servidor dedicado puede ser elevado, y la administración de un servidor web requiere conocimientos técnicos avanzados.
